Element Display Properties
Display: Inline & Display: Inline-Block

Html elements are like unique video game characters within an Html document. Each element has unique attributes and abilities. Html elements have attributes that are set by default.

If you know any Html you will know some of these elements. I donâ€™t know much HTML myself, but here we go:

We have <div> tags, we have our heading tags <h1> - <h6>, paragraph tags <p>, <span> tags and <img> tags. We call each element when we need to format our content. The <H1> tags make some text bolder, <img> lets us add an image. <div> tags create a container taking up the full width of screen, <span> tags create containers that take up only as much space as they need. Each tag has inherent attributes. Like video game characters some are loud and powerful, like <div> tags, taking up the full width on the page, some are more peaceful like <span> tags, that will sit side by side.

But we can specify and change some of these inherent properties. One of them is the *display* property. By default, this attribute is set to block or inline. display: block elements always start on a new line and take up the full width available (they stretch out to the left and right as far as they can). An inline element does not start on a new line and only takes up as much width as necessary. Note: We can set the width and height of display: block elements but not of display: inline elements

These elements are have the display property block by default:

<div>
<h1> - <h6>
<p>
<form>
<header>
<section>

These ones are inline by default:

<span>
<a>
<img>

We can change the attributes of these elements when we want, we can change a <div> element to display: inline or span element to display: block. There is also a third option where we change the display property to display: inline-block

Ultimately this post was supposed to be about display: inline and display:inline-block - basically, display: inline is helpful as it lets elements sit horizontally next to each other, but with this attribute, we cannot set the width and height. Display: inline-block - lets elements sit next to each other and lets us set the width and height properties. Its the best of both worlds.
