Problem Solving Techniques

Getting blocked on a simple problem

A simple problem I got stuck on was taking a sentence and capitalising the first letter of every word. When you say this problem out loud it seems quite simple - but getting a fucntion to do it, as a learning developer, provided some challenges.

For this problem I had to learn about a built in 'application' in Javascript - we call them methods. This method could search through a bunch of words and perform an action on each word, exactly what I needed to do this challenge! So I was learning about this method and I couldn't seem to get it to work, it was the first step in the solving the problem and I felt like I was already failing. I thought the problem was that I was initalsing the 'method' in the wrong way, but It turns out I needed to convert the information I was feeding into the method, and that most of my code was ok.

It was like I was trying to jam a cassette tape in a CD player. It was never going to work.

The thing is I knew the information I was 'giving' to the method was wrong but I was to flussterd about it not working to realise this was the problem. In the end I asked my tutor about the issue, he pointed out that the input was the wrong 'format' and that sent me on my way to solving the problem.

I felt overwhelmed by this problem, instead of focusing on the immediate 'slice' of the problem in front of me, I kept 'zooming' out and getting overwhelmed by thinking about the problem in it entirety.

Solving a problem in an elegant way

This was a small stepping stone on the way to solving a larger problem, but I was proud of how I solved it. I had a bunch of test scores belong to a student. This was inside a javascript document, and the scores and the name of the student were in different places. Instead of typing out the scores in the new place, I simply referenced or created a little window beside the students name showing where the tests score were.

To solve this problem I just tried something.

Through this challenge I learnt that almost everything in a javascript file can be linked to another, we can draw a little thread from one keyword to a another so that now it references it, like a detective solving a murder on a corkboard.
